# PrepNest Frontend

PrepNest is a React + TypeScript study platform focused on USAT and HAT preparation.

## Tech Stack

- Vite
- React 18 + TypeScript
- Tailwind CSS
- Radix UI primitives
- React Router
- React Query

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
